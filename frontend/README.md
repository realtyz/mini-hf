# mini-hf 项目介绍

## 项目概述

mini-hf 是一个模型缓存仓库系统，用于在局域网内缓存和管理 Huggingface 或 Modelscope 上的数据集和模型。

## 核心功能

1. **模型/数据集缓存**
   - 将 Huggingface 或 Modelscope 上的模型和数据集缓存到 S3 存储
   - 支持自动拉取和存储远程资源

2. **局域网访问加速**
   - 局域网用户通过设置 `HF_ENDPOINT` 环境变量即可快速下载
   - 避免重复从外网下载相同资源，节省带宽

3. **任务队列管理**
   - 采用任务排队模式处理下载请求
   - 避免多任务并行下载时的带宽争抢问题
   - 确保资源有序分配

4. **安全隔离**
   - 屏蔽用户直连 Huggingface 的访问
   - 减少潜在的信息安全风险
   - 提供统一的下载入口

## 使用场景

适用于需要在内网环境中频繁下载 Huggingface 模型和数据集的团队或组织，提供高速、安全、可控的资源访问方式。
